
# アカウントの作成
## Docker
- Docker（ドッカー）は開発環境を共有する仕組みです。
- 複数人でシステムを開発するときに、それぞれのソフトウェアのバージョンを揃えることができます。
- 下記のウェブページからアカウントを作成しておいてください。
  - https://www.docker.com/

## GitHub
- GitHub（ギットハブ）はプログラムを共有する仕組みです。
- 複数人でシステムを開発するときに、他の人が開発している部分を最新に保つことができます。
- 下記のウェブページからアカウントを作成しておいてください。
  - https://github.com/
